# COMPILER
[+] Kelebihan
- siap untuk dijalankan
- lebih cepat
- source code tidak terlihat

[+] Kekurangan
- tidak lintas platform
- tidak fleksibel
- langkah tambahan


# INTERPRETER
[+] Kelebihan
- lintas platform
- fleksibel
- mudah untuk diuji dan mudah untuk di modifikasi

[+] Kekurangan
- Butuh interpreter
- lebih lambat
source code mudah di akses

contoh bahasa :
COMPILER : C,C++, Objective-C
INTERPRETER : javascript,PHP 
hybrid : java,VB.NET,Python
