jika kita memnggunakan sebuah Operator di atas, nilai yang sebelumnya
kita simpan di dalam sebuah variabel tidak akan hilang. tapi, akan di lakukan
Operasi terlebih dahulu.

Operator penugasan      |       sama saja dengan
    x += y                         x = x + y
    x -= y                         x = x - y
    x *= y                         x = x * y
    x /= y                         x = x / y
    x %= y                         x = x % y


Operator Perbandingan / comparasion :
Operator Perbandingan       |       keterangan
    ==                              sama dengan
    !=                              tidak sama dengan

    strict adalah Operator identitas.
    ===                             strict sama dengan
    !==                             strict tidak sama dengan

    >                               lebih besar dari
    <                               lebih kecil dari
    >=                              lebih besar sama dengan 
    <=                              lebih kecil sama dengan
Perbandingan ini akan menghasilkan boolean " True Atau False"

 == dan === : apa bedanya? 

 == --> dia tidak memandang tipe data
 contoh : 10 == "10" hasil; True
 === --> dia memandang tipe data
 contoh : 10 === "10" hasil; False
